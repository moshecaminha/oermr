⬢ Dados Abertos

API pública para desenvolvedores e pesquisadores.

Todos os indicadores estão disponíveis via API REST e GraphQL. Download em CSV, JSON e Parquet. Conformidade total com a Lei de Acesso à Informação.

  • 14 endpoints temáticos cobrindo todos os indicadores
  • Atualização automática diária via Apache Airflow
  • Histórico completo desde 2010
  • Webhooks para notificação de novos dados
  • SDK em Python, R e JavaScript
  • Sem necessidade de autenticação para consultas básicas
